== Access the Application ==

1. Get the frontend URL:
{{- if eq .Values.frontend.service.type "NodePort" }}
   minikube service {{ include "todo-app.fullname" . }}-frontend -n {{ .Release.Namespace }}
{{- else }}
   kubectl port-forward svc/{{ include "todo-app.fullname" . }}-frontend {{ .Values.frontend.service.port }}:{{ .Values.frontend.service.port }} -n {{ .Release.Namespace }}
   Then open: http://localhost:{{ .Values.frontend.service.port }}
{{- end }}

2. Check pod status:
   kubectl get pods -n {{ .Release.Namespace }}

3. View logs:
   kubectl logs -l app.kubernetes.io/component=frontend -n {{ .Release.Namespace }}
   kubectl logs -l app.kubernetes.io/component=backend -n {{ .Release.Namespace }}

== Cleanup ==

   helm uninstall {{ .Release.Name }} -n {{ .Release.Namespace }}
